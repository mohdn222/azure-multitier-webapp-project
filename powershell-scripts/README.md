# Azure Multi-Tier Web Application Project

## Overview
This project demonstrates how to design and deploy a highly available multi-tier web application on Microsoft Azure.

The architecture includes:

- Virtual Network with Subnets
- Application Gateway
- VM Scale Sets with Autoscaling
- Azure SQL Database
- Azure Storage
- Azure Bastion
- Monitoring with Azure Monitor

## Technologies Used

- Microsoft Azure
- Azure PowerShell
- GitHub
- Virtual Machine Scale Sets
- Azure Application Gateway
- Azure SQL Database
